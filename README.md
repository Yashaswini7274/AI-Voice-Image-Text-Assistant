# AI-Voice-Image-Text-Assistant
Converting speech to text quickly and accurately is a growing challenge in digital communication.
# 🎤 Voice-to-Text + 🖼️ AI Image Generator

## 📌 Overview
This project is an AI-powered application that converts voice input into text using speech recognition and generates corresponding images using a text-to-image model.

The system integrates Whisper (for speech-to-text) and Stable Diffusion (for image generation) to create a seamless voice-to-image pipeline.

---

## 🚀 Features
- 🎤 Voice input using microphone
- 📝 Speech-to-text conversion using Whisper
- 🖼️ AI image generation using Stable Diffusion
- 🌐 Interactive web interface using Gradio
- ⚡ Real-time processing and output display

---

## 🛠️ Technologies Used
- Python
- Whisper (OpenAI)
- Gradio
- Hugging Face API
- Stable Diffusion XL
- LiveKit API (connection testing)
- Requests & Pillow

---

## ⚙️ How It Works
1. User records voice input using microphone
2. Whisper model transcribes speech into text
3. The transcribed text is sent to a text-to-image model
4. The model generates an AI image based on the prompt
5. Output is displayed in a web interface
